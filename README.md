# 🏥 Projeto ARKUS - Sistema de Gestão Logística para Saúde

O **ARKUS** é um sistema Full-Stack desenvolvido para otimizar o fluxo de coletas laboratoriais e a organização logística de atendimentos técnicos. O objetivo principal é permitir o cadastro estruturado de demandas e a visualização rápida por data.

## 🚀 Funcionalidades Principais
- **Cadastro Manual:** Registro de coletas com nome, endereço, turno e horário.
- **Importação de Lote (CSV):** Possibilidade de subir uma lista de pacientes via arquivo CSV para agilizar o processo.
- **Filtro Dinâmico:** Visualização das demandas filtradas por data de agendamento.
- **Visualização Profissional:** Tipos de coleta padronizados (Coleta de Sangue e Retirada de Material).

## 🛠️ Tecnologias Utilizadas
- **Backend:** Python com framework Flask.
- **Frontend:** HTML5, CSS3 (Design Geométrico) e JavaScript (Fetch API).
- **Banco de Dados:** MySQL para persistência de dados.
- **Integração:** Manipulação de arquivos CSV com suporte a encoding `utf-8-sig`.

## 📂 Estrutura do Repositório
- `/backend`: Contém o servidor Flask (`app.py`).
- `/front-end`: Contém a interface do usuário (`index.html`).
- `database.sql`: Script para criação da estrutura do banco de dados.
