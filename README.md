# 🏥 Projeto ARKUS – Sistema de Roteirização Domiciliar

O **ARKUS** é um sistema inteligente focado na otimização de fluxos logísticos para coletas domiciliares. O objetivo principal é permitir o cadastro estruturado de demandas e a definição das rotas mais eficientes, reduzindo o tempo de planejamento e garantindo agilidade no atendimento ao paciente.

## 🚀 Funcionalidades AC1 (Concluído)

- **Módulo de cadastro manual:** Desenvolvido para a gestão de demandas emergenciais e encaixes de última hora. Garante flexibilidade ao fluxo, permitindo que novas solicitações sejam integradas em tempo real.
- **Importação de planilha:** Projetado para a importação massiva de dados. Esta abordagem elimina erros de digitação, garante a integridade das informações e otimiza o tempo de preparação operacional.
- **Filtro dinâmico:** Interface que permite o controle das demandas usando filtros por data e funcionalidade de exclusão para cancelamentos, facilitando a definição de prioridades.

## 🛠️ Tecnologias Utilizadas
- **Backend:** Python 
- **Frontend:** HTML, CSS e JavaScript
- **Banco de Dados:** MySQL (altere as credenciais de usuário e senha se necessário)
