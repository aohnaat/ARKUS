# 🧭 ARKUS

Sistema de organização e roteirização de atendimentos.

---

## 📌 Sobre o projeto
O ARKUS é um sistema desenvolvido com o objetivo de auxiliar na organização de atendimentos técnicos,
permitindo o cadastro e a visualização das informações de forma estruturada.

Este projeto está sendo desenvolvido como atividade acadêmica, com entregas incrementais.

---

## ⚙️ Tecnologias utilizadas
- Front-end: HTML e CSS
- Back-end: Python
- Banco de dados: SQLite

---

## 🗂️ Estrutura do projeto

```arkus/
├── backend/
│ └── app.py
├── frontend/
│ ├── index.html
│ └── style.css
└── README.md
```

---

## 🚀 Funcionalidades
- Estrutura inicial do sistema
- Separação entre front-end e back-end
- Preparação para cadastro de atendimentos

---

## 📅 Planejamento
O desenvolvimento do projeto está dividido em etapas:
- AC1: Estrutura inicial do projeto
- AC2: Cadastro e gerenciamento de atendimentos
- AC3: Organização e simulação de rotas
- Entrega final: Sistema completo
