# 🧭 ARKUS

Sistema de organização e roteirização de atendimentos.

---

## 📌 Sobre o projeto
O ARKUS é um sistema desenvolvido com o objetivo de auxiliar na organização de atendimentos técnicos,
permitindo o cadastro e a visualização das informações de forma estruturada.

Este projeto está sendo desenvolvido como atividade acadêmica, com entregas incrementais.

---

## ⚙️ Tecnologias utilizadas
- Front-end: HTML e CSS
- Back-end: Python
- Banco de dados: SQLite
